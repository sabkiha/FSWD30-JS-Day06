Day 4 | Challenge

Challenge Description:

Many people love games of chance and especially the thrill of the "roll of the dice". For this Challenge, create a page with a two-sided layout for two players based on the idea of "roll of the dice".

When the page loads, a prompt method should appear to take a string (the name for each player), then display their names in the header of the page, for example "Thomas vs. Martin".

In each half of the layout, place a button that calls a function. This function should create at least three random numbers and show their sum. Next, display the numbers separately and visually appealing in the each players half, so that the player can see what they "rolled". Once the both players rolled, announce the Winner based on the sum! Do your best to create a rich game-design using CSS.


Feel free to go beyond our coding requirements and also demonstrate your creativity, simply don't feel limited in any aspect!



Consider the following programming principles while solving the Challenge:

  • Use of best practices in semantic elements, code indentation and comments
  • Give your page a responsive layout with at least two breakpoints
  • Correct use of array methods
  • Correct use of at least one type conversion
  • Correct use of named and anonymous functions
  • Correct use of attributes with functions
  • Declaring variables as global or local when appropriate
  • Best practices in naming variables
  • Manipulating the DOM with methods like .getElementByID() and properties like .innerHTML


  • Bonus:
    • Use SASS to style your page
    • Create a function that is called each time the page refreshes and generates a random background color.
